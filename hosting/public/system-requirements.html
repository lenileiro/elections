System requirements

These project was developed on Ubuntu 18.04 (LinuxPlus 2019)

To install ubuntu on your work computer you would need to:

  • Download Ubuntu from there official site (Ubuntu 2019)
  • Install the iso file on a flash disk
  • Boot the workstation computer from the USB disk
  • Follow the installation guide on screen to complete installation process
