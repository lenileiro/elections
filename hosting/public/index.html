1 Introduction

This is a project an elections application where by citizen, politician and electral commission can perticipate on the election process. The main of the project to demostrate use of the following tectnologies.

  • Python Flask framework
  • Relational Database - Postgres
  • Container orchestration - Docker
  • Project Management - Pivotal Tracker
  • Continous Intergration - Travis CI
  • Test Driven Development - Coveralls
  • Git Version Control - Github
  • Application deployment - Heroku
  • Code Documentation - Rmarkdown
  • API Documentation - API Blueprint
